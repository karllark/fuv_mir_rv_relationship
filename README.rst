Code for FUV to MIR Spectroscopic R(V) Relationship paper
=========================================================

Routines for FUV to MIR spectroscopic R(V) depedent relationship for
Milky Way dust extinction work.
This paper is Gordon, et al. (in prep).

In Development!
---------------
